 # TCP/IP
   - Прикладной уровень (Application Layer) 
     - HTTP, RTSP, FTP, DNS
   - Транспортный уровень (Transport Layer) 
     - TCP, UDP, SCTP, DCCP 
   - Межсетевой уровень (Internet Layer)
     - IP, ICMP, IGMP,  ARP
   - Канальный уровень (Network Access Layer)
     - Ethernet, IEEE 802.11 WLAN, SLIP, Token Ring, ATM, MPLS
